=================================

   Dance×MixerDLC移行ツール

=================================
≪！注意！≫
・DLCを継続して使用する場合は必ずこのツールを正しい順序で
　使用してください。
・適用順序を間違えるとDLCが使用できなくなります。
=================================
この度はDance×MixerのDLC移行ツールを
ダウンロード頂き、誠にありがとうございます。

このフォルダに収められているツールを使用する事で
DLCの使用環境を移行する事が可能です。

また万が一Dance×Mixerの再インストールが必要になる前に
一度このツールを使用してDLC使用環境のバックアップを
作成する事をお勧め致します。

□使用手順□
≪必ず各ツールを使用する場合は右クリックから表示される
　「管理者権限で起動」を使用し起動するようにして下さい。≫

1）DLCを使用可能な環境で、各エディタのあるフォルダ
（例：C:\Program Files\DanceMixer）に『DlcMaker』を置いて実行します。
 ⇒同フォルダ内に『DlcPac』というファイルが出来れば成功です。

2）DLCを移行させたい環境にDance×Mixerをインストールし、
　アップデートデータ(http://dancemixer.jp/patch/UPDATE.zip)を適用させ
　各ファイルを最新版(http://dancemixer.jp/patch/DMEXE.zip)に書き換えておきます。

3）手順1）で出来た『DlcPac』ファイルと『DlcMove』を、
　手順2）で準備した環境各エディタのあるフォルダへ配置し、『DlcMove』を実行します。
　追加パック用のフォルダにDLCが作成されていれば成功です。
　(『DlcPac』ファイルは成功時には自動的に削除されます。)
 
このツールを使用する際、
 *）移行元と移行先両方の環境に
 　・CharEditor.exe
 　・DanceEditor.exe
 　・DressEditor.exe
 　・DanceEditorOld.exe
 　・追加パック用のフォルダ（基本的にインストール時に作成されるので敢えて削除したりしていなければ気にする必要はありません）
 これらの必要なファイルがどれか一つでも足りない場合はエラーになり実行されません。
 *)DLC移行後、エディタを他のバージョンで上書してしまうとDLCの使用が不可能になります。
 *)手順1)を実行時DLCが使用できない環境であった場合、手順3)で最新バージョン(アップデートデータ以外)が適用されている場合、
 　DLCの移行は失敗となります。
